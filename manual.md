# Jenkins Project Setup Guide

1.  **Jenkins Setup**: Access your Jenkins dashboard.
2.  **Manual Way**: Begin manual configuration of a new job.
3.  **Enter a Project Name**: Provide a unique name for your project.
4.  **Select Freestyle Project**: Choose the "Freestyle project" option.
5.  **Click OK**: Confirm your selection to proceed to configuration.
6.  **Go to Source Code Manager**: Navigate to the "Source Code Management" section.
7.  **Choose Git**: Select **Git** and enter the URL of your remote repository from GitHub.
8.  **Go to Build Step**: Scroll down to the "Build Steps" section.
9.  **Add Build Step**: Click the "Add build step" dropdown menu.
10. **Execute Batch Command**: Select **"Execute Windows batch command"**.
11. **Enter Command**: Input your execution command (e.g., `python script.py`).
12. **Save and Build**: Save the job configuration and click **"Build Now"** to run it.
